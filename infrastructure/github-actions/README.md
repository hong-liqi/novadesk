# GitHub Actions

Workflows ativos ficam em `.github/workflows/` (requerido pelo GitHub).

| Workflow | Arquivo | Descrição |
|----------|---------|-----------|
| CI | `ci.yml` | lint, typecheck, test, build |
| Dependency Review | `dependency-review.yml` | Auditoria de dependências em PRs |

Documentação completa: [docs/06-DevOps.md](../../docs/06-DevOps.md)
