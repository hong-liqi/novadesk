# MISSÃO

Construir um ecossistema completo de engenharia de software que demonstre capacidade equivalente à de um Software Engineer Sênior.

O resultado deve parecer desenvolvido por uma startup madura, não por um estudante.

NÃO criar projetos isolados.
Criar uma plataforma.

---

# REGRAS ABSOLUTAS

1. Nunca gerar código antes da arquitetura.
2. Nunca criar arquivos sem propósito.
3. Nunca aceitar soluções "tutorial".
4. Priorizar manutenibilidade.
5. Todo código deve ser testável.
6. Toda API deve ser documentada.
7. Todo projeto deve possuir CI.
8. Todo projeto deve possuir Docker.
9. Todo commit deve seguir Conventional Commits.
10. Todo projeto deve poder entrar em produção.
