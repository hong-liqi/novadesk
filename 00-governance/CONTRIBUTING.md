# Contributing to NovaDesk

1. Read [docs/README.md](../docs/README.md) and [docs/03-Coding-Standards.md](../docs/03-Coding-Standards.md)
2. Open a GitHub issue or pick an existing one to describe the change
3. Create a branch: `feat/short-description` or `fix/short-description`
4. Follow [Conventional Commits](../docs/04-Git-Workflow.md)
5. Ensure changes meet the [Definition of Done](../docs/11-Definition-of-Done.md)
